This game was made for the August 2014 Ludum Dare Challange with the theme Connected Worlds. It was made for the 72 hour challenge, but life took its toll and the game wasn't completed in time to be submitted. The main game was still completed in 72 hours, as a personal challenge. Extra time has been put in to polish the game.

It is built entirely by two people, bootstrut and tropvache. All ideas, graphics, music, and code have come through us. Javascript makes up most of the guts and brains of this game, and we use DOM manipulation to make the everything move and react to a player. This is an HTML5 game, and it has been optimised for both desktop and mobile devices! If you run into any issues with the game play, please tweet at us or start a ticket on github.

Use the arrow keys on your keyboard or swipe the screen in the direction you want to move.

You control one character, but there is another little guy who follows in the opposite fashion. When you move up, it moves down.

You want to collect points, and avoid bombs. Your score will be saved to your browser so you can gloat to your friends. Let them try and fail to beat your highscore.
